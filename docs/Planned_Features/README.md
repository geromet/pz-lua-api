# Planned Features

Features that are designed or spec'd but not yet implemented.

## File naming
`FEAT-NNN-short-description.md`

## Template
```
# FEAT-NNN: Short Title

**Priority:** High | Medium | Low
**Area:** Extractor | Frontend | Nav | Source Viewer | ...
**Depends on:** (other features or data requirements)

## Goal
One-sentence summary of what this adds for the user.

## Design
Detailed description of the intended behaviour and UI.

## Implementation Notes
Extractor changes needed, frontend changes, data shape, etc.

## Open Questions
- ...
```
