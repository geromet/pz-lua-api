# Skills

Instruction files for Claude covering how to handle specific recurring scenarios in this project. A skill is a canned procedure — when the trigger condition is met, follow the steps exactly.

**When to read this folder:** When a recurring task comes up that has a known correct procedure.
**When to write here:** When the user corrects Claude on how to do something for the second time — that's a signal it should be a Skill.

_(empty — add skill files as recurring scenarios are identified)_

## Template

```markdown
# SKILL: Short Name

**Trigger:** When [condition that should invoke this skill]

## Steps
1. ...
2. ...

## Why
Rationale, or past incident that motivated this skill.
```
